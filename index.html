Engraved

Engraved is a Discord Bot that logs all the messages the users write on your Discord server, it also offers a dashboard where a user can login and check all the messages that have been written, you can also search messages from a certain username or a certain date. It uses MongoDB as it's database and uses Discord's OAuth2 API for authentication.
